# Codes
## Textbild
- Neuer Text. Ein Zeilenumbruch ensteht erst durch eine Leerzeile, oder 
doppelten Space. 
Diese drei Zeilen stehen zusammen.

- '#' = '<H1>'
- '*' oder '_' = kursiv
- '**' oder '__' = fett
- '***' oder '___' = fett + kursiv
- '~~' = durchgestrichen
## Listen
- '1.' = nummerierte Liste
- '"2 spaces"' = bullet-Unterliste
- '*' '-' '+' = bullet-Liste
